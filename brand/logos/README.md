# Logos

Coloque aqui os arquivos oficiais (SVG/PNG) da marca. Por padrão, tudo em `logos/*` (exceto este `README` e `.gitkeep`) está no [`.gitignore`](../.gitignore) para evitar vazar ativos. Se a política do repositório for versionar as logos, remova a regra `brand/logos/*` do `.gitignore` e comite com cuidado.

Atualize `assets.logo_svg` em `tokens.json` (cópia a partir de `tokens.json.example`) se o caminho for outro.
