# Exemplos de deck (JSON)

Estes arquivos são **genéricos** e podem ir para o Git. Servem como ponto de partida para o agente copiar estrutura e layouts.

- `board_expansao.json` — deck completo de exemplo (vários layouts).

Para usar a paleta do trabalho, adicione no mesmo JSON algo como `"brand_path": "../brand.json"` (caminho relativo à pasta deste arquivo).
