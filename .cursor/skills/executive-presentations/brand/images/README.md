# Imagens aprovadas pela marca

- Formatos: JPG ou PNG.
- Preferência: **horizontal** (16:9 ou 3:2), largura mínima **1600px** para não pixelar em telas grandes.
- Uso: no `deck.json`, `image_path` relativo à pasta do arquivo JSON (ex.: se o deck está em `brand/examples/`, use `"../images/hero.jpg"`).

Arquivos reais nesta pasta **não** são versionados (exceto este README e `.gitkeep`).
