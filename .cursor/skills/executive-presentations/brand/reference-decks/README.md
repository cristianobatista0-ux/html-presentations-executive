# Decks de referência

Coloque aqui **PDF** ou **PPTX** de apresentações que o time considera “padrão ouro” (tom, densidade, hierarquia).

O agente pode **ler** estes arquivos (via ferramentas do Cursor) para alinhar estilo e estrutura — não são embedados no HTML automaticamente.

Arquivos reais nesta pasta **não** são versionados (exceto este README e `.gitkeep`).
