# Logos

- Preferência: **SVG** vetorial (escala nítida em qualquer tamanho).
- Alternativa: **PNG** com fundo transparente, @2x (ex.: 512×512 ou largura mínima 400px).
- Nome sugerido: `logo.svg` ou `logo.png` — referencie em `brand.json` como `"logo_path": "logos/logo.svg"` (caminho relativo à pasta `brand/`, onde está o `brand.json`).

Arquivos reais nesta pasta **não** são versionados (exceto este README e `.gitkeep`).
